# KineticTwin – SmartFootAI
## Overview
KineticTwin – SmartFootAI is an AI-driven spatial intelligence platform designed to predict mall footfall and identify optimal locations for piezoelectric energy-harvesting tile installation. The system leverages machine learning techniques to analyze visitor patterns and support data-driven infrastructure planning.
## Problem Statement
Installing piezoelectric tiles across an entire mall is costly and inefficient. Mall administrators require a reliable method to identify high-footfall zones where energy generation potential is maximized.
## Solution
SmartFootAI predicts visitor density based on factors such as:

* Hour of the day
* Day of the week
* Mall zone
* Weather conditions
* Special events

The predicted footfall data is then used to identify critical high-traffic zones suitable for piezoelectric tile deployment.

## Machine Learning Model
*Initially we created a manual dataset which lacked the patterns between the variables also we used label encoding for encoding the cateogorical variables and Algorithm used was linear regression.Due to this the Model's accuracy was low (BASELINE ACCURACY: 26.36%).So we improved our model's performance by using the following techniques:
* Algorithm: Random Forest Regressor
* Preprocessing: One-Hot Encoding
* Evaluation Metric: R² Score
* Final Accuracy: ~90%
 we focused heavily on perfecting our Random Forest architecture and data engineering blocks. We then utilized AI collaboration to rapidly develop and deploy a responsive Streamlit dashboard interface, turning our backend logic into a live, interactive, production-ready platform.
